# 🤝 Contributing to Cognify AI

Thank you for considering contributing to Cognify AI! 🎉

## How Can I Contribute?

### For Beginners:
- Fix typos in README or comments
- Add more RSS sources to `news_generator.py`
- Improve UI design

### For Experienced:
- Add dark mode toggle
- Add PDF export feature
- Add voice input support

## How to Submit Changes?

1. Fork the repository
2. Clone to your computer (`git clone ...`)
3. Make your changes
4. Submit a Pull Request

## Code Guidelines

- Use 2 spaces for indentation
- Use meaningful variable names
- Test in Chrome and Firefox before submitting

## Questions?

Open an issue on GitHub

Thank you! 🌟
